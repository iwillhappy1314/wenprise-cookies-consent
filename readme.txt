A simple and lightweight cookie consent notice plugin for WordPress.

== Description ==

Wenprise Cookies Consent is a lightweight plugin that adds a customizable cookie consent notice to your WordPress website. It helps you comply with cookie laws and GDPR requirements.

= Features =

* Lightweight and fast
* Customizable notice text
* Two button options: Accept (30 days) and Close (1 day)
* Mobile responsive design
* Translation ready
* Clean and modern design

= Languages =

* English (Default)
* Simplified Chinese (简体中文)

== Installation ==

1. Upload the `wenprise-cookies-consent` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Cookies Consent to customize your notice text

== Frequently Asked Questions ==

= How can I customize the notice text? =

Go to Settings > Cookies Consent in your WordPress admin panel. You can edit the notice text there.

= How long does the cookie last? =

* When visitors click "Accept", the notice will not show for 30 days
* When visitors click "Close", the notice will not show for 1 day

= Can I translate the plugin? =

Yes, the plugin is translation-ready. You can use tools like Poedit to translate the plugin into your language.

== Privacy Policy ==

This plugin only sets a cookie to remember user's choice about the cookie notice. It does not collect, store, or share any personal data.

The cookie set by this plugin:
* Name: wprs_cc_notice_accepted
* Purpose: Remember user's choice about cookie notice
* Duration: 30 days (Accept) or 1 day (Close)
